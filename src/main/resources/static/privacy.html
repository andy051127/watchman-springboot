개인정보처리방침

최종 업데이트: 2026년 1월 1일

1. 수집하는 개인정보

회원가입 시 다음 정보를 수집합니다.

  • 이메일 주소 (로그인 식별자로 사용)
  • 닉네임
  • 비밀번호 (암호화하여 저장)

서비스 이용 중 자동 생성되는 정보:

  • 공부 세션 기록 (집중 시간, 집중률, 포인트)
  • 프로필 사진 (직접 업로드 시)

2. 카메라 데이터

카메라 영상은 서버로 전송되지 않습니다. 모든 얼굴 인식 분석은 사용자의 브라우저(기기) 내에서만 처리되며, 영상 데이터는 어디에도 저장되지 않습니다.

  • 영상 녹화 없음
  • 서버 전송 없음
  • 얼굴 데이터 저장 없음
  • 분석 결과(집중/비집중 상태)만 로컬에 기록

3. 개인정보의 보유 및 이용기간

회원 탈퇴 시 모든 개인정보와 공부 기록이 즉시 삭제됩니다. 현재 서비스는 모든 데이터를 사용자 기기의 로컬 저장소(localStorage)에만 저장하며, 별도의 서버 보관이 없습니다.

4. 이용자의 권리

이용자는 언제든지 다음 권리를 행사할 수 있습니다.

  • 개인정보 열람 및 수정 (마이페이지에서 직접 변경)
  • 회원 탈퇴 (마이페이지 → 위험 구역 → 회원 탈퇴)
  • 카메라 권한 철회 (브라우저 설정에서 언제든지 가능)

5. 문의

개인정보 관련 문의는 문의하기 페이지를 통해 연락해 주세요.
